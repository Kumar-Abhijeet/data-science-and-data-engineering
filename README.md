## 🚀 Getting Started

### 1. Clone the Repository
```bash
git clone https://github.com/Kumar-Abhijeet/data-science-and-data-engineering.git
cd data-science-and-data-engineering
```

### 2. Create a Virtual Environment
```bash
# Windows
python -m venv venv
venv\Scripts\activate

# macOS / Linux
python3 -m venv venv
source venv/bin/activate
```

### 3. Install Dependencies
```bash
pip install -r requirements.txt
```

### 4. Start Learning
Open [`learning-modules/phase-01-python-foundations/`](learning-modules/phase-01-python-foundations/) and begin with Day 1.
