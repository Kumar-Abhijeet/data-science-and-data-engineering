# Data Science and Data Engineering

This repository is a beginner-friendly starting point for learning:

- Data Science fundamentals
- Databricks
- PySpark

## Suggested Learning Path

1. **Python Basics**
   - variables, functions, loops, and data structures
2. **Data Analysis**
   - NumPy, Pandas, data cleaning, and visualization
3. **SQL Basics**
   - filtering, joins, aggregations, and window functions
4. **PySpark Fundamentals**
   - Spark DataFrames, transformations, actions, and Spark SQL
5. **Databricks**
   - workspace basics, notebooks, clusters, jobs, and Delta Lake
6. **Projects**
   - end-to-end mini projects using real datasets

## Recommended Next Steps

- Set up Python and Jupyter Notebook locally
- Install PySpark
- Create a Databricks Community Edition account
- Start with one beginner project and build consistently
