# Phase 5 — Machine Learning

**Duration:** ~6–8 weeks  
**Goal:** Build end-to-end machine learning models and deploy them.  
**Prerequisites:** Phase 2 — Data Analysis, Phase 4 — Statistics

---

## 🎯 What You Will Learn

| Module | Topics |
|--------|--------|
| Supervised Learning | Linear/Logistic Regression, Decision Trees, Random Forest, XGBoost |
| Unsupervised Learning | K-Means, DBSCAN, PCA, dimensionality reduction |
| Feature Engineering | Encoding, scaling, feature creation, selection |
| Model Evaluation | Cross-validation, confusion matrix, ROC-AUC, RMSE |
| Pipelines | sklearn Pipelines, ColumnTransformer, GridSearchCV |
| Deployment Basics | joblib/pickle, FastAPI serving, streamlit demo |
