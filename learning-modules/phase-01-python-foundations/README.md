# Phase 1 — Python Foundations

**Duration:** ~4–6 weeks  
**Goal:** Build strong Python fundamentals using real-world projects.

---

## 🎯 What You Will Learn

| Day | Module | Key Skills |
|-----|--------|-----------|
| 1 | Variables & Data Types | int, float, str, bool, type(), input() |
| 2 | Operators & Expressions | Arithmetic, comparison, logical, string ops |
| 3 | Lists & Tuples | Indexing, slicing, list methods, mutability |
| 4 | Dictionaries & Sets | Key-value pairs, set operations |
| 5 | Conditional Logic | if/elif/else, nested conditions |
| 6 | Loops | for, while, range(), enumerate(), zip() |
| 7 | Functions Basics | def, parameters, return, scope |
| 8 | Functions Advanced | *args, **kwargs, lambda, map, filter |
| 9 | OOP Part 1 | Classes, objects, __init__, attributes, methods |
| 10 | OOP Part 2 | Inheritance, encapsulation, polymorphism |
| 11 | File Handling | open(), read/write/append, CSV, JSON |
| 12 | Exception Handling | try/except/finally, custom exceptions |
| 13 | Modules & Packages | import, pip, standard library |
| 14 | Virtual Environments | venv, pip freeze, requirements.txt |

## ✅ Phase 1 Exit Criteria

Before moving to Phase 2, you should be able to:
- [ ] Write a Python script from scratch without help
- [ ] Explain the difference between mutable and immutable types
- [ ] Build a class with inheritance and use it in a project
- [ ] Read and write CSV/JSON files
- [ ] Handle exceptions gracefully
- [ ] Set up a virtual environment and manage dependencies
