# Phase 6 — Advanced Data Science

**Duration:** ~6–8 weeks  
**Goal:** Master advanced DS techniques including NLP, time series, and deep learning basics.  
**Prerequisites:** Phase 5 — Machine Learning

---

## 🎯 What You Will Learn

| Module | Topics |
|--------|--------|
| NLP Basics | Tokenisation, TF-IDF, bag-of-words, text classification |
| Time Series | Decomposition, ARIMA, Prophet, forecasting |
| Deep Learning Basics | Neural networks, backprop intuition, Keras |
| LLM Basics | Transformers, prompting, API integration (OpenAI, HuggingFace) |
| MLOps Intro | Experiment tracking (MLflow), model versioning, CI/CD basics |

---

## 🏗️ Phase 6 Projects

1. **Sentiment Analysis** — Product review classifier
2. **Sales Forecasting** — Time series forecasting with Prophet
3. **Chatbot** — Intent classification + LLM-powered responses
4. **AI Analytics Assistant** — LLM + data analysis
