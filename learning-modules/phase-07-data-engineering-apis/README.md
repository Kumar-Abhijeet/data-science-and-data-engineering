# Phase 7 — Data Engineering + APIs

**Duration:** ~4–6 weeks  
**Goal:** Build automated data pipelines and integrate external APIs.  
**Prerequisites:** Phase 3 — SQL + Python

---

## 🎯 What You Will Learn

| Module | Topics |
|--------|--------|
| APIs & JSON | REST APIs, HTTP methods, authentication, JSON parsing |
| ETL Pipelines | Extract-Transform-Load patterns, data validation |
| Airflow Basics | DAGs, operators, scheduling, monitoring |
| Spark Basics | RDDs, DataFrames, PySpark basics |
| Cloud Basics | AWS S3, GCS, Azure Blob — reading/writing data |
| Docker Basics | Containers, images, Dockerfile, compose |

---

## 🏗️ Phase 7 Projects

1. **Automated ETL Pipeline** — API → transform → database
2. **API Data Ingestion** — Pull, validate, and store external data
3. **Real-time Analytics Pipeline** — Streaming data basics
