# Phase 4 — Statistics & Mathematics

**Duration:** ~3–4 weeks  
**Goal:** Build the statistical foundations required for machine learning.  
**Prerequisites:** Phase 2 — Data Analysis

---

## 🎯 What You Will Learn

| Module | Topics |
|--------|--------|
| Probability | Events, sample space, conditional probability, Bayes' theorem |
| Distributions | Normal, binomial, Poisson, uniform, skew |
| Hypothesis Testing | t-test, chi-square, p-value, confidence intervals |
| Correlation & Regression | Pearson/Spearman correlation, simple & multiple linear regression |
| Linear Algebra Basics | Vectors, matrices, dot product, eigenvalues (intuition) |

## 💡 Teaching Approach

Statistics is taught visually and intuitively:
- Every concept paired with a real business scenario
- Code using `scipy.stats`, `numpy`, `matplotlib`
- No heavy mathematical notation until intuition is built
