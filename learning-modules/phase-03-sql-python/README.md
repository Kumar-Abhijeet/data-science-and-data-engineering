# Phase 3 — SQL + Python

**Duration:** ~3–4 weeks  
**Goal:** Master SQL from basics to advanced, and integrate it with Python.  
**Prerequisites:** Phase 1 — Python Foundations

---

## 🎯 What You Will Learn

| Module | Topics |
|--------|--------|
| SQL Basics | SELECT, WHERE, ORDER BY, LIMIT, DISTINCT |
| SQL Intermediate | JOINs (INNER, LEFT, RIGHT, FULL), UNION |
| SQL Advanced | CTEs, Window Functions, Subqueries, Indexes |
| Query Optimisation | Execution plans, indexes, partitioning |
| Python + SQL | SQLite, SQLAlchemy, psycopg2, pandas read_sql |
