# Phase 2 — Data Analysis

**Duration:** ~4–6 weeks  
**Goal:** Master data analysis using NumPy, Pandas, and visualization libraries.  
**Prerequisites:** Phase 1 — Python Foundations

---

## 🎯 What You Will Learn

| Module | Topics |
|--------|--------|
| NumPy | Arrays, vectorized operations, broadcasting, indexing |
| Pandas | Series, DataFrame, indexing, groupby, merge, reshape |
| Data Cleaning | Missing values, duplicates, outliers, data types |
| EDA | Descriptive stats, distributions, correlation, patterns |
| Visualization | Matplotlib, Seaborn, Plotly — charts, heatmaps, dashboards |
| Statistics Basics | Mean, median, std, percentiles, distributions |
