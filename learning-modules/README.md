# Learning Modules

This folder contains all structured learning paths organised by phase.

## Phases

| Folder | Phase | Topics |
|--------|-------|--------|
| `phase-01-python-foundations/` | Python Foundations | Variables, Loops, Functions, OOP, File Handling |
| `phase-02-data-analysis/` | Data Analysis | NumPy, Pandas, EDA, Visualization, Statistics |
| `phase-03-sql-python/` | SQL + Python | SQL Basics–Advanced, Python DB Connections |
| `phase-04-statistics-mathematics/` | Statistics & Maths | Probability, Distributions, Regression, Linear Algebra |
| `phase-05-machine-learning/` | Machine Learning | Supervised, Unsupervised, Pipelines, Deployment |
| `phase-06-advanced-data-science/` | Advanced Data Science | NLP, Time Series, Deep Learning, LLMs, MLOps |
| `phase-07-data-engineering-apis/` | Data Engineering + APIs | ETL, Airflow, Spark, Docker, Cloud |
| `phase-08-powerbi-analytics/` | Power BI + Analytics | Python–Power BI, KPIs, Dashboards |

## How to Use

1. Work through phases in order.
2. Each module folder contains:
   - `README.md` — overview, objectives, and session map
   - `theory.md` — concept notes with examples
   - `practice.py` — guided coding exercises
   - `exercises.py` — self-practice tasks (try before looking at solutions)
   - `solutions/` — reference solutions
   - `interview_questions.md` — common interview questions for the topic
